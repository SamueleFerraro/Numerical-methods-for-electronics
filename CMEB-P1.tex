\section{Point a)}
Let us define the primal formulation of the problem:
\begin{equation}
    \begin{cases}
        &-(\mu(x)u'(x))' + \sigma u(x) = f(x)\\
        &u(a) = u(b) = 0
    \end{cases}
\end{equation}

The flux $J(x)$ can be defined by using the constitutive law:

\begin{equation}
    J(x) = - \mu(x) u'(x)
\end{equation}

To find the discrete flux density $J_h(x) \in Q_h(a,b)$, I expand the equation using the weak formulation and testing it using the basis functions defined in \eqref{BasisFunctionPrimal}. \\
\begin{equation}
    \int_{a}^{b} J(x) \psi(x) \, dx = -\int_{a}^{b} \mu(x)u'(x) \psi(x) \, dx
\end{equation}
First of all, let us compute the right hand side of the equation:
\begin{equation}
    \int_{a}^{b} J(x) \psi(x) \, dx = \int_{x_{i-1}}^{x_i} J(x) \cdot 1 \, dx = \int_{x_{i-1}}^{x_i} J(x) \, dx
\end{equation}
while the left hand side:
\begin{equation}
    -\int_{a}^{b} \mu(x)u'(x) \psi(x) \, dx = -\int_{x_{i-1}}^{x_i} \mu(x)u'(x) \, dx
\end{equation}
